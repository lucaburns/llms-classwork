# llms-classwork
Classwork for a Special Topics Math Course, LLMs, taught by Dr. Michael Ivanitsky (GitHub: mivanit).

## Transformer from Scratch

This project taught us about the LLM architecture, tokenization, training loop, and text generation.
This was a group project, but I include only my contributions in this repo.
The group project can be found at https://github.com/bguevel/LLM1.

## Logit Lens and Tuned lens

This project taught us about logit lenses and tuned lenses which are forms of linear probes used in mechanistic interpretability to decode hidden layers into vocabulary distribution. More details can be found in the logit-tuned-lenses folder.
